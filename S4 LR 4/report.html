Лабораторная работа

Тема работы: Доверительный интервал

Цель работы: Изучить методы построения доверительных интервалов для среднего значения, доли и дисперсии генеральной совокупности

Задание 1.

Постановка задачи: Азимут оси ВПП измерен в 4 приема. Результаты измерений представлены в таблице. Требуется оценить точность результата измерения.

ixixi - x(xi - x)^2
125° 17' 30''-15''225
225° 17' 45''0''0
325° 18' 15''30''900
425° 17' 30''-15''225

Решение:

Начальные данные: n = 4, степень свободы = 3, доверительная вероятность = 0,95.

Среднее значение x̄ = 25° 17' 45''.

Дисперсия S2 = 450, стандартное отклонение S = 21,21.

Среднеквадратическое отклонение центра распределения = 10,61.

Коэффициент Стьюдента t = 3,18.

Предельная погрешность ≈ 34.

Доверительный интервал: от 25° 17' 11'' до 25° 18' 19''.

Задание 2.

Постановка задачи: Проверен расход энергии в 10 квартирах. Данные: 125, 78, 102, 140, 90, 45, 50, 125, 115, 112.

Определить доверительный интервал с надежностью 0,95.

Решение:

N = 70, n = 10, γ = 0,95.

t = 2,26.

Среднее значение x̄ = 98,2, стандартное отклонение S = 32,145.

Предельная погрешность Δ = 21,27.

Доверительный интервал: (76,93; 119,47).

Задание 3.

Постановка задачи: Определить объем выборки при N = 1000 и надежности 0,95.

Решение:

t = 1,96.

Повторная выборка: n = 384.

Бесповторная выборка: n = 277.

Задание 4.

Постановка задачи: В партии 5000 изделий, проверено 400, из них 300 высшего сорта.

Решение:

t = 1,96, W = 0,75.

Интервал (повторная): (0,7076; 0,7924).

Интервал (бесповторная): (0,7093; 0,7907).

Задание 5.

Постановка задачи: Найти объем выборки для 10000 банок при ошибке 0,05 и надежности 0,9995.

Решение:

t = 3,5, W = 0,5.

Повторная выборка: n = 1225.

Бесповторная выборка: n = 1091.

Задание 6.

Постановка задачи: Найти доверительный интервал для среднего квадратического отклонения.

Решение:

Среднее значение = 0,3.

Дисперсия S2 = 0,01158.

Интервал для дисперсии: (0,006; 0,034).

Интервал для СКО: (0,077; 0,184).

Вывод

В ходе лабораторной работы были построены доверительные интервалы для среднего значения, доли и среднего квадратического отклонения для повторной и бесповторной выборок.
